# Better Crafter (for Fabric)

![Crafter block](src/main/resources/assets/bettercrafter/icon.png)

[![Static Badge](https://img.shields.io/badge/Version-1.0.0--1.21.4-green)](CHANGELOG.md)

A small, simple but effective mod that improves the use of Crafter block. Forked from [BetterCrafter](https://github.com/Fern-Aerell/BetterCrafter)

## Required
- Fabric API ([Modrinth] (https://modrinth.com/mod/fabric-api)/ [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api))

## Features
### New Crafter Mechanism
- The Crafter block will only produce its crafting output if the quantity of items in each crafter slot is more than 1.
